Declaración de Accesibilidad

Compromiso con la inclusión y accesibilidad para todos

Nuestro Compromiso

En el ministerio del Evangelista Jhonatan Danny Rivas, estamos comprometidos con hacer que nuestro sitio web sea accesible para todas las personas, incluyendo aquellas con discapacidades. Creemos que el mensaje del evangelio debe estar disponible para todos, sin barreras.

Estándares de Accesibilidad

Nuestro sitio web sigue las pautas de accesibilidad al contenido web (WCAG 2.1) nivel AA. Estas pautas establecen los estándares internacionales para hacer que el contenido web sea más accesible para personas con discapacidades.

Características de Accesibilidad

Modo Oscuro/Claro

Puedes alternar entre tema oscuro y claro usando el botón de cambio de tema.

Texto Redimensionable

Puedes ampliar el texto usando las funciones de zoom del navegador.

Navegación por Teclado

Todas las funciones son accesibles mediante teclado (Tab, Enter).

Estructura Semántica

Usamos encabezados para facilitar la navegación con lectores de pantalla.

Texto Alternativo

Todas las imágenes incluyen descripciones para lectores de pantalla.

Subtítulos en Videos

Los videos incluyen subtítulos cuando están disponibles.

Limitaciones Conocidas

A pesar de nuestros esfuerzos, algunas áreas del sitio pueden tener limitaciones:

  • Algunos videos antiguos pueden no tener subtítulos disponibles
  • El contenido incrustado de YouTube tiene su propia configuración de accesibilidad
  • El traductor de Google puede tener limitaciones en su interfaz
  • Algunos documentos PDF pueden no ser completamente accesibles
